# Reliability-Aware Ensemble-CAM: A Proposed Extension

**Author:** Kayes Bin Yousuf | Incoming MS ECE, Clarkson University

## Overview
A proposed reliability extension to the Ensemble-CAM face
presentation attack detection (PAD) pipeline, adding
post-hoc calibration and uncertainty quantification.

## Contributions
1. Temperature Scaling - post-hoc calibration via NLL grid search
2. Conformal Prediction - uncertainty-aware prediction sets (90% coverage)
3. Uncertainty-Guided CAM - heatmap scaled by calibrated confidence
4. Retention Comparison - extends Table 3 of the paper
5. Distribution Shift Robustness - cross-domain evaluation
6. Overhead Analysis - computational cost breakdown
7. Failure Case Analysis - when reliability extension helps/hurts
